SITE PARA GABRIELA - VERSÃO SEM DATA COMEMORATIVA

Esse projeto foi atualizado para ser um presente romântico de qualquer dia, sem mencionar Dia das Mulheres, aniversário ou outra celebração específica.

COMO PUBLICAR:
1. Extraia o arquivo ZIP
2. Envie todos os arquivos para o repositório
3. Vá em Settings > Pages
4. Em Build and deployment, escolha Deploy from a branch
5. Selecione a branch main e a pasta /root
6. Salve

MÚSICA:
- Essa versão usa a música do YouTube
- Vídeo usado: https://www.youtube.com/watch?v=WRxwtV_x8nM
